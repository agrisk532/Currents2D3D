Program features

 

This program is an attempt to calculate from the first principles and render the superconducting electric currents and resulting magnetic fields of a high-Tc crystal in it's ground state.

 

The main features of the program:

  • Include any size YBaCuO cluster in the calculation. However, the CPU time will increase accordingly.
  • Dope the system by removing (or adding) electrons.
  • Specify applied magnetic field value.
  • Both CuO planes of the unit cell are included in 3D calculation. For 2D rendering only one CuO plane.
  • Calculate and render in 2D or 3D space the 1D Cu-Cu bond electric currents.
  • In 3D space render the applied, strengthened, weakened and total magnetic field, produced by 1D currents.
  • Calculate and render in 3D space the true 3D electric currents.
  • Multiple document interface allows to work with any number of simultaneously opened windows from the same application.
  • Include different atoms in rendering.
  • 3D volume ray cast alpha compositing, maximum intensity projection, isosurface, streamline rendering options.

 

At present several restrictions are present:

  • Only the YBaCuO crystal structure can be calculated.
  • No periodic boundary conditions.
  • No lattice defect option support.
  • Only CuO planes are included in the electric current calculation. No other atom layers.
  • Single gauge origin for magnetic field calculation, in the symmetry center of a cluster.
  • Fixed parameters for colors, atom radiuses, grid settings.
  • No phonon influence can be calculated.
  • Single thread program architecture
